## Overview

Development of a Property Listing Web Application.

## Live Demo

- Unavailable due to the deletion of the college student's AWS account

## Key Features

- **Comprehensive Property Management**: Manage property listings to maintain an organized and up-to-date catalog.
- **Agent Profile Management**: Manage agent profiles and view all listings linked to agents.
- **Advanced Filtering Options**: Filter property listings by property type, price range, number of bedrooms, and other criteria. Filter agent profiles by agent type for quick searches.
- **Robust Input Validation**: Ensure data accuracy and security with server-side validation for all user inputs.
